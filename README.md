# Смета.AI

Прототип AI-сметчика для подрядчиков ИЖС и сметчиков-фрилансеров. Заполняешь форму — получаешь черновик сметы за 5 минут вместо 4–8 часов в Excel.

> Цель прототипа — валидация продукта через customer development. Если вы фаундер, который проверяет гипотезу, или сметчик/подрядчик, готовый поделиться обратной связью, — пишите.

## Демо

Открыть `index.html` в браузере. Никакой сборки не нужно — это нативный HTML/JS прототип.

Альтернативно: [GitHub Pages версия](https://your-username.github.io/smeta-ai/) (после форка).

## Что внутри

- `index.html` — UI прототипа: форма ввода параметров дома + панель результата
- `app.js` — упрощённый расчётный движок: формулы объёмов, региональные коэффициенты, итог
- `sample-smeta.pdf` — пример выходного документа, который получает пользователь после оплаты
- `docs/` — материалы по продукту: позиционирование, customer-development скрипты, план продвижения

## Технологический стек прототипа

- Vanilla HTML + CSS + JavaScript (ES6). Никаких зависимостей.
- Открывается напрямую с диска или хостится на любом статике (GitHub Pages, Vercel, Netlify).

Продакшен-стек, к которому стек прототипа эволюционирует:
- Next.js 14 + TypeScript + Tailwind для фронта
- Python FastAPI для расчётного движка
- Supabase (Postgres + Auth + Storage) для пользователей и проектов
- Anthropic Claude API + Yandex Vision OCR для парсинга ТЗ и пояснительных записок
- ЮKassa для оплат
- Vercel + Railway для хостинга

## Точность расчётов в прототипе

Прототип использует упрощённую модель на базе **6 регионов** и **типовых работ** для одно- и двухэтажных домов из газобетона, кирпича, бруса и каркасных конструкций. Точность ±10–15% от реального рынка 2026.

В продакшен-версии:
- 100+ позиций работ и материалов по нормам ФЕР
- Региональные коэффициенты для 85 субъектов РФ
- Цены материалов парсятся еженедельно с Лемана ПРО, Петрович и региональных баз
- Калибровка по 200+ реальным сметам, выполненным сметчиками-консультантами

## Roadmap

**v0 — этот прототип (сделано)**
- Базовая форма ввода
- Расчётный движок с 6 регионами
- UI для демонстрации на интервью

**v1 — MVP (8–10 недель)**
- Регистрация и личный кабинет (Supabase)
- Платежи через ЮKassa
- Генерация PDF и Excel со сметой
- 100+ позиций ФЕР, актуальные цены материалов
- Парсинг ТЗ через Claude API (текст → параметры формы)

**v2 — Зрелый продукт (4–6 мес)**
- Брендированные КП для клиента (шаблоны для подрядчика)
- OCR-парсинг PDF-проектов
- Интеграция с поставщиками стройматериалов (актуальные цены через API)
- Подписочная модель, командные аккаунты

**v3 — Экспансия (12+ мес)**
- Поддержка ИЖС-норм Казахстана и Беларуси
- Распознавание чертежей через computer vision
- Голосовой ввод параметров
- Интеграция с банками ИЖС-ипотеки
